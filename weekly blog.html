Timeline of Lessons Learnt

(you can click on the title links to see more)

Week 1

Introduction, FabLab Safety, Tools

We learnt about Fab Lab safety specifically for those equipments like example Laser Cutting & 3D Printing found in SP.


Certification

After the e-packages we also have quiz to show that we understand and have to pass above 80%

Week 2

Software tools, HTML & CSS, Website

We have learnt some simple html and also basic of a weboage through our online lesson that is recorded for us. We learnt about the essential software tools for a digital fabricator and how to document our work.

Week 3

Personal Website Development, HTML & CSS

We learnt some simple css in class as such on how to change the layout of the website to the preference of our choice and not only that we get to make it more asethetic way.

GitHub

We learnt what Github is used for and also the basics of creating a repository and uploading our codes through there

Week 4

Computer Graphics, Computer Aided Design

We were introduced to computer graphics (eg. raster and vector) and did basic modeling using extrude in Autodesk Fusion 360.

Week 5

2D Drawing and 3D Modeling

We learn to used the basic functions of Fusion 360 and create a 2d drawing to 3d modeling like we did some exercise like Name Tag, Lego Brick, Extrudes & Planes, Mug with Handle and Hex Nut


Week 6

3D Modelling & 3D Printing

We used more in depth of modeling functions in Fusion 360 such as revolve and sculpting. We also modelled a chess piece and handphone stand. Nevertheless, we get to used the 3D Printer to print out our chess piece

Week 7

Computer Controlled Cutting & Laser Cutting

We learnt about preparing our designs for cutting operations by using Fusion 360 to get the vector profiles then check out DXF files in Inkscape. We used a name tag and laptop stand to practice.

Week 8

Laser Cutting Quiz & Practical Demonstration

We have to go through a online quiz and practical test before utilizing this laser cutting machines and is important to make sure that all of us able to used it without any supervision.

Week 9

Parametric box Assignment

We are to design any type parametric box that have lid/door and drawer made using lazer cutting, and one that is easily modified using the use of parameters.Last but not least able to fit and no loose parts.

Week 12

Electronic Embedded Programming

We are to leant about electricity and common electrical components like switches, resistors, LEDs etc. We started working on simple circuits involving the breadboard and Arduino Uno on TinkerCAD.

Week 13

Basic output devices

We started working with the Arduino software and TinkerCAD to do start working on coded programs and circuits.

Week 14

Basic input devices

We learnt about the Digital I/O and Analog I/O and how to implement them in circuits that included push buttons and LEDs. We learn about the different sensors and actuators that are available and how to program some of them to work.

Week 15

Module Project

We started working on our module project and brain storming some ideas. I then decided on what project to do and submitted an overview of what our projects would cover and what components we would need.
