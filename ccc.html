Computer Controlled Cutting Introduction

  • For Computer Controlled Cutting, it works on a 2D profile which provides an outline of the cut so the advantages are:
    • The cut is precise with little wastage of material.
    • Allows positioning to prevent wastage.
    • Repeatability

    Therefore, vector file formats are needed to begin laser cutting, these are DXF (Data eXchange Format), PDF (Portable Document Format), SVG (Scalable Vector Graphics). For my assignments I would be using Inkscape to produce my DXF files for printing.
