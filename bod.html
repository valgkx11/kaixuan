Introduction to basic output devices

  • Basic output devices: Introduction
    • The concept of output device can be understood by an example that if an appliance works on the values coming from the sensor so the appliance will be an output device and the sensor will be an input device.
    • Some of the output devices such as LEDs, Motors, LCDs, Relays, Seven segment displays, Speaker and Buzzer

Digital Output

  • Digital output exists when a voltage is applied to a pair of digital output leads.
  • This voltage can be sourced (sinked) by internal or external sources.
  • Therfore, there can be pulse output when the device is active.
  • The simplest example of digital output is a light bulb.
    • The output will turn on the light bulb or turn off the light bulb.

Analog Output

  • Same as analog input,analog output also produces continuous results
  • If the input is changed, the output is also changed almost instantaneously.
  • The implementation of analog output can be seen from the meters such as pressure meter, water level meter, etc.

Actuator Output

  • An actuator is an output device for the conversion of supply energy into useful work.
  • The output signal is controlled by the control system, and the actuator responds to the control signals via the control element.
    • Like example servo motor.
