Digital Fabrication & Prototyping Fundamentals (EP1000) is an elective module offered at Singapore Polytechnic. In this module you will learn how to use Digital Fabrication techniques to design and create a prototypes. Like example Computer Aided Designing techniques using 3D printing, lazer cutting and work documentation like making website through HTML and CSS will be taught.
